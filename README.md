# Mahesh Rajendra - Portfolio Website

A modern, dark-themed portfolio website for Mahesh Rajendra, a Data Analytics & Visualization Professional. Built with HTML5, CSS3, and JavaScript, featuring a sleek design with pastel color highlights and responsive layout.

## üåü Features

### Design & UI
- **Modern Dark Theme** with pastel accent colors (mint, peach, lavender, cyan, pink)
- **Responsive Design** optimized for desktop, tablet, and mobile devices
- **Smooth Animations** and transitions throughout the site
- **Gradient Backgrounds** with floating animated orbs
- **Typography** using Inter font family for clean, modern look

### Sections
1. **Hero Section** - Professional introduction with call-to-action buttons
2. **About** - Personal bio and educational background
3. **Services** - Professional services offered
4. **Portfolio** - Project showcase with hover effects
5. **Experience** - Work history with timeline layout
6. **Skills** - Technical skills organized by category
7. **Contact** - Contact information and contact form

### Interactive Features
- **Mobile Navigation** with hamburger menu
- **Smooth Scrolling** between sections
- **Form Validation** with user-friendly notifications
- **Scroll Animations** using Intersection Observer
- **Hover Effects** on cards and interactive elements
- **Active Navigation** highlighting current section
- **Typing Animation** for hero title
- **Parallax Effects** on background elements
